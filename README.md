# Task Organizer Project

context organizer for different data to be visualized with widgets.

### stack

- Node.js
- Express
- MongoDB / Mongoose
- Docker

### Steps

1. npm init -y
2. npm install express mongoose dotenv cors
3. Create server.js file
4. Create .env file for environment variables
5. Setup MongoDB connection using Mongoose
6. Create models for tasks
7. Create routes for CRUD operations
8. Setup Express server
9. Test API using http client
10. (Optional) Dockerize the application
11. (Optional) Deploy to cloud service like Heroku or AWS
